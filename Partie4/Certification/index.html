Fonctionnalites

Service de partage de fichiers

Ce service vous permet de partager des fichiers sur plusieurs appareils differents en utilisant le meme compte. Vous avez droit à 1TB de stockage.

Service de serveur web

Ce service vous permet de beneficier d'un hebergement web avec nom de domaine offert. Vous pouvez posseder jusqu'a 3 sites web ou dans la limite du stockae disponible (100GO)

Service de suite collaborative

Ce service vous permet de beneficier d'une suite de logiciels (traitement de texte, Diaporama, tableur) heberge dans la cloud. L'acces aux fichiers peux etre autoriser a plusieurs utilisateurs en meme temps.
